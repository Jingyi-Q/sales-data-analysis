# Sales Data Analysis Project

## Project Background
This project analyzes a superstore sales dataset to answer three business questions:
1. What is the total revenue?
2. Which product category generates the highest sales?
3. How does sales fluctuate over time?

The insights can help managers with inventory planning, marketing strategies, and category focus.

## Tools Used
- Python (pandas, sqlite3, matplotlib)
- Jupyter Notebook
- SQL

## Repository Structure
- `sales_analysis.ipynb` - Jupyter notebook containing all code (SQL + pandas) and outputs
- `monthly_trend.png` - Monthly sales trend chart
- `README.md` - Project description

## Key Findings
1. **Total Sales**: $2.26 million
2. **Sales by Category**:
   - Technology: $827,456 (highest)
   - Furniture: $728,659
   - Office Supplies: $705,422
3. **Monthly Trend**: Sales range from $10,000 to $27,000 per month, with peaks in August 2015 and September 2018. No strong seasonality detected.
